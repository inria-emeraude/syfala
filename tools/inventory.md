## Guideline

Serial number logic for homemade PCBs:\
**XXNYYAAA**

**XX**: PCB Type (see below)\
**N**: Revision\
**YY**: Year of manufacture\
**AAA**: Unique ID

| PCB Type           | Value  |
| :---------------- | :------: |
| Controller board | 01 |
| TDM Board unpopulated | 02 |
| TDM board factory populated | 03 |
| Motherboard unpopulated | 04 |
| Motherboard factory populated | 05 |
